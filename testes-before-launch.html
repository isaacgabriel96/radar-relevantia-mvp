⚠️ Limitação: JavaScript não funciona em emails

Gmail, Outlook e Apple Mail bloqueiam JavaScript — então os botões acima funcionam aqui no preview, mas não no email real enviado.

Opções para implementar de verdade:

  1. Sem botão no email — deixar o texto bem destacado (já está) e o usuário copia manualmente. Simples, sem custo.
  2. Botão como link externo — o botão no email abre radar.../copiar?v=SENHA, uma mini-página que copia e fecha. Funciona em todos os clientes.
  3. Versão web do email — adicionar "Ver no navegador" no topo; a versão web tem JS e os botões funcionam normalmente.

Qual você prefere? Posso implementar qualquer uma das opções.
